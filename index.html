HTML

HTML es un lenguaje de marcado que se utiliza para crear paginas web. Es el lenguaje de marcado estandar para la creacion de documentos en la World Wide Web. HTML se compone de una serie de elementos que se utilizan para estructurar y presentar el contenido de una pagina web. Estos elementos incluyen encabezados, parrafos enlances, imagenes, tablas y mcuhos otros HTML es un lenguaje muy flexible y se puede utilizar para crear una amplia variedad de sitios web, desde simples paginas estaticas hasta aplicaciones web complejas.
